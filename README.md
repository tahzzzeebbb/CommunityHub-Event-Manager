# CommunityHub-Event-Manager
A full-stack Community Event Management system built with PHP, JavaScript Fetch API, and HTML5/CSS3. Features secure user authentication, role-based access for organizers/members, dynamic event creation, real-time RSVPs, and automated user notifications.

🚀 Key Features

• Secure Authentication & Registration: Implements modular login and registration workflows with structured inputs for full names, emails, community identification, and distinct system roles.
• Role-Based Access Control (RBAC): Differentiates permissions dynamically between standard 'Members' and 'Organizers/Admins' to restrict sensitive functionalities like event creation.
• Dynamic Event Management: Form processing portals allowing authorized supervisors to seamlessly provision new events with designated titles, descriptions, timelines, geographic locations, and seat capacities.
• Interactive RSVP Submissions: Fully integrated client-side Fetch API operations handling asynchronous asynchronous payload distributions for user RSVPs and dynamic attendee list representations.
• Live Notifications: A centralized update board rendering chronological notification objects regarding upcoming schedules, location modifications, and event deadlines.

🛠️ Tech Stack Used
• Backend: PHP (Session tracking, role management, and modular API endpoints)
• Frontend: HTML5, CSS3 (Scoped component styling), Vanilla JavaScript (Asynchronous Fetch API & JSON payloads)
